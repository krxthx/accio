TrustAgentAI – Cryptographic receipts for MCP tool calls (non-repudiation layer)
Hacker News · 2026-03-18
TrustAgentAI adds a non-repudiation layer to the Model Context Protocol (MCP) by generating cryptographic receipts for AI agent tool calls. The system addresses MCP's accountability gap by providing tamper-evident records with cryptographic proof of who authorized each action and the outcome. This enables verifiable audit trails for high-stakes operations like financial transactions, replacing editable logs and unverified timestamps.
OmniInfer: Cross-platform LLM inference framework for edge deployment
GitHub · 2026-03-20
OmniInfer is a high-performance cross-platform inference engine for LLMs and vision-language models (VLMs), designed for local deployment across Android, iOS, macOS, Windows, and Linux. It abstracts model compilation and hardware adaptation, supporting multiple backends including llama.cpp (GGML-based) and a native OmniInfer engine. The framework offers OpenAI API compatibility while keeping all inference local for privacy, serving as the core engine for Omni Studio.

A better method for identifying overconfident large language models
MIT News AI · 2026-03-19
MIT researchers introduced a new uncertainty quantification method that compares a target LLM's responses to responses from similar LLMs to detect confidently incorrect outputs. Their cross-model disagreement approach was combined with self-consistency measures into a total uncertainty metric, which outperformed existing methods across 10 realistic tasks including question-answering and math reasoning. This addresses a critical limitation: traditional methods only measure self-confidence rather than actual prediction accuracy, making them ineffective at identifying models that are confidently wrong.
Diffusion-Guided Semantic Consistency for Multimodal Heterogeneity
arxiv · 2026-03-19
This arXiv paper (2603.19337) addresses federated learning challenged by non-IID (non-independent and identically distributed) client data, which degrades global model performance in multimodal perception settings. The research identifies that conventional methods fail to address underlying semantic discrepancies between clients, leading to suboptimal performance. The proposed approach uses diffusion-guided semantic consistency to handle multimodal heterogeneity across federated clients.
