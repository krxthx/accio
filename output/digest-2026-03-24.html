GoSkill: Goal-driven execution helper for long-running tasks with explicit success criteria
GitHub · 2026-03-20
GoSkill is a Python library that wraps tasks with goal-driven execution logic, enabling long-running operations to continue until explicit success criteria are met or timeout occurs. It supports defining goals, criteria (e.g., "compile": "0 errors", "test": "100% pass"), max_hours, and max_attempts parameters. The library provides two return modes: run() for raw results and run_with_result() for structured output including success status, attempts count, and criteria reports. Unlike simple function calls that return immediately, GoSkill persistently attempts execution and validates against defined acceptance criteria.

MARLIN (Multi-Agent Reinforcement Learning for Incremental DAG Discovery) is a novel approach for uncovering causal structures from observational data in the form of directed acyclic graphs. The method tackles the efficiency limitations of current reinforcement learning approaches, making it viable for real-time and online application scenarios where rapid causal inference is essential.
